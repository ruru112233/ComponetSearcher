## 使い方
①コンポーネント名欄に検索したいコンポーネントの名称を入力

②検索オプション、その他オプションを設定（表示されていない場合は「検索オプションを展開する」にチェックを入れる）

③検索にヒットすると、オブジェクト名称が表示され、そのボタンをクリックすると対象のオブジェクトが選択される
